# ExamplePack

This is an example content pack demonstrating the modding structure for GL2 Engine.

## Structure

- `items.json` - Item definitions
- `blocks.json` - Block definitions
- `biomes.json` - Biome definitions
- `rooms.json` - Room definitions

## Adding Your Own Pack

1. Create a new folder in `/Mods/` with your pack name
2. Add JSON files following the same structure
3. Packs are loaded in alphabetical order (later packs override earlier ones)

## JSON Format

See the individual JSON files for examples of the expected format.
